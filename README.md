# Bluestock Mutual Fund Analytics Capstone

## Project Overview

This project focuses on building a complete mutual fund analytics pipeline using Python, SQLite, SQL, and Power BI.

### Day 1 Deliverables

* Project setup and folder structure
* Data ingestion for 10 datasets
* Data quality assessment
* Fund master exploration
* AMFI code validation
* Live NAV data collection using MFAPI

### Tech Stack

* Python
* Pandas
* NumPy
* SQLite
* SQLAlchemy
* Power BI
* Git & GitHub
